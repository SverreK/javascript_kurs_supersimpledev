# SuperSimpleDev JavaScript kurs
Dette repoet inneholder kode og øvelser jeg har laget mens jeg følger JavaScript-kurset til SuperSimpleDev. Formålet med repoet er å lære JavaScript fra bunnen av og bygge et solid fundament for frontend- og webutvikling.

# Status
8 av ca. 22 timer fullført

Per nå dekker koden i dette repoet:
* Intro og JavaScript Basics
* Numbers and Math
* Strings
* HTML og CSS Review + console.log
* Variables
* Booleans og If-Statements
* Functions
* Objects
* Document Object Model (DOM)
* HTML, CSS og JavaScript samlet

# Kursoversikt
Igjenstående temaer i dette kurset er:
* Arrays og Loops
* Advanced Functions
* Amazon Project og Intro til Git
* Modules
* External Libraries
* Testing og Testing Frameworks
* Object-Oriented Programming
* Intro til Backend og Asynkron JavaScript

# Teknologier brukt
* JavaScript
* HTML
* CSS
* DOM API
* Git og GitHub

# Mål
Målet mitt med å lære JavaScript er:
* Å forstå JavaScript i dybden
* Kunne bygge interaktive nettsider
* Bruker JavaScript med HTML, CSS og React
* Få eit solid grunnlag for frontend. og fullstack-utvikling
